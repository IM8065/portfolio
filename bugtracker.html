Bugtracker Application

Created a bug tracker application using ASP.NET Core 3.0 MVC, where projects can be made with tickets assigned to it indicating bugs. The ticket is customizable with a user being able to specify the status (open/closed) and severity of the problem. The application also has Authentication and Authorization with different user roles, Administrator and a regular user. The Administrator has more privileges than the regular user such as deleting other users and creating and assigning roles.

Project: Bugtracker Application

Technologies: HTML, CSS, JAVASCRIPT
